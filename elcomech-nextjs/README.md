# Elcomech Systems Website

A production-ready Next.js website for Elcomech Systems — a Bengaluru-based industrial engineering company.

## Tech Stack

- **Framework**: Next.js 14 (Pages Router)
- **Styling**: Tailwind CSS 3
- **Animations**: Framer Motion
- **Font**: Montserrat (Google Fonts)
- **Forms**: React Hook Form + Zod
- **Carousel**: Embla Carousel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

| Page | Route |
|------|-------|
| Home | `/` |
| About | `/about` |
| Products & Services | `/products` |
| Product Detail | `/products/[slug]` |
| FAQ | `/faq` |

## ICT/FCT Fixture Wizard

The PCB Test System and Testing Service product pages include a guided 7-step ICT/FCT fixture project wizard based on the official Elcomech fixture requirements checklist. Click "Start ICT/FCT Project" on either product page to launch it.

## Build for Production

```bash
npm run build
npm run start
```
